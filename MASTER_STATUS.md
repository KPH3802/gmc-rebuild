                }

Local-workflow note (non-authoritative; supports §8 step 6). Before opening any implementation PR, run `pre-commit run --all-files` on a **clean local checkout** (i.e. with no uncommitted local autoformat or autofix edits) so that lint, formatting, type-check, and secret-detection failures surface in the same form CI will report. Opening an implementation PR that passes locally only because of uncommitted autoformat edits has, in the recent P3-03 sequence, required a follow-on remediation PR (PR #44 / PR #45); the requirement to pass `pre-commit run --all-files` on a clean checkout is intended to avoid that pattern.
