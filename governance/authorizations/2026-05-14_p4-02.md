  - `__init__.py` — test package marker (empty body or minimal module docstring only). No runtime side effects on import. No re-export of any merged Phase 3 fixture from any runtime path.
  - `test_composed_fixture.py` — focused, deterministic tests for the **composed** in-memory fixture, asserting:
    - structural conformance of each of the three in-memory fakes (`InMemoryHeartbeat`, `InMemoryKillSwitch`, `InMemoryReconciliation`) to its corresponding `runtime_checkable` Protocol declared in `src/gmc_rebuild/risk/` (`HeartbeatProtocol`, `KillSwitchProtocol`, `ReconciliationProtocol`);

  - The test module avoids `pytest.raises` and fixture-typed parameters to remain compatible with the mypy-strict pre-commit hook (the same pattern P3-03 / P3-04 / P3-05 tests already follow per `tests/test_risk_interfaces.py`).
