## Allowed Scope

The implementation PR may include exactly:

- A new subpackage at `src/gmc_rebuild/simulation/` containing:
  - `src/gmc_rebuild/simulation/__init__.py` — exports exactly `SimulatedIntent`, `SimulationBoundary`, `SimulationBoundaryError`, and `SimulationLane`. The package is **not** re-exported from `src/gmc_rebuild/__init__.py`.
  - `src/gmc_rebuild/simulation/_boundary.py` — declares (a) a closed `SimulationLane` `StrEnum` whose only authorized member is `LOCAL_ONLY`; (b) a frozen `SimulatedIntent` dataclass with three fields (`lane: SimulationLane`, `intent_id: str`, `created_at: str`) and a `build(*, lane, intent_id, created_at)` classmethod that accepts a UTC `datetime` and emits an ADR-004 `Z`-suffixed string via `gmc_rebuild.risk.to_utc_string`; (c) a `SimulationBoundary` class whose only public operation `propose(*, intent, verdict)` returns the supplied intent unchanged if and only if the boundary's lane matches the intent's lane and the supplied `gmc_rebuild.runtime.SafetyVerdict` is `clear`, and raises `SimulationBoundaryError` (a `ValueError` subclass) with the verdict's blocker tuple in the message on any other condition; (d) the `SimulationBoundaryError` class itself.
  - The module's permitted imports are exactly `__future__`, `dataclasses`, `datetime`, `enum`, `gmc_rebuild.risk` (for `to_utc_string` only), and `gmc_rebuild.runtime` (for `SafetyVerdict` only). The module must not import `os`, `socket`, `requests`, `urllib`, `http`, `threading`, `asyncio`, `subprocess`, `sqlite3`, `pickle`, `shelve`, `ssl`, `smtplib`, `ftplib`, or any other I/O / network / persistence module.
